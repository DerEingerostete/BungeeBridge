# BungeeBridge
An API that makes it easier to use the PluginMessagingChannel

### Maven:
```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>
```
```xml
<dependency>
    <groupId>com.github.DerEingerostete</groupId>
    <artifactId>BungeeBridge</artifactId>
    <version>1.2.6</version>
</dependency>
```
